Задача

С нуля собрать маркетинг-функцию для сложного B2B Enterprise-продукта и вывести компанию на рынок.

Что сделал

  • Позиционирование, ICP/персоны, ABM, контент-матрица, PR, ивенты, перформанс, CRM-воронки.
  • Новый сайт (SEO-ядро, аналитика), демо-видео, кейсы, TG-канал, вебинары/конференции.
  • Сквозная аналитика (Power BI / GA / Calltouch / Roistat), AmoCRM, пайплайн NQL→MQL→SQL.
  • Пресейл: TCO/ROI для заказчиков; PR tier-1 (РБК, Forbes); статусы МТК Skolkovo, Sber 500 Bootcamp, ФРИИ Sprint 7.
